📚 Jogo de Reconstrução da Biblioteca (Minigames Educativos)
Projeto desenvolvido utilizando a Unity como parte de um circuito de ciências, com foco em educação ambiental e conscientização sobre o bioma Cerrado.
🎯 Contexto
Este projeto foi desenvolvido em grupo para um circuito de ciências com temática voltada ao Cerrado e sustentabilidade.
A equipe foi composta por três integrantes de diferentes áreas:
Desenvolvimento de jogos (responsável pelo jogo)
Design
Outra área complementar
Cada integrante contribuiu com seu conhecimento para a construção do projeto, integrando elementos físicos e digitais apresentados no evento.
🌱 Objetivo
O jogo tem como objetivo conscientizar sobre questões ambientais, como poluição, preservação da fauna e queimadas, utilizando uma abordagem interativa.
A biblioteca presente no jogo representa simbolicamente a biblioteca escolar, sendo restaurada conforme o jogador completa desafios relacionados ao meio ambiente.
🎮 Mecânica principal
O jogador interage com partes da biblioteca danificada, onde cada elemento desbloqueia um minigame.
Ao concluir os desafios, a biblioteca é restaurada progressivamente até sua reconstrução completa.
🧩 Minigames
🦊 1. Alimentação da fauna
Controle de uma raposa
Coleta de alimentos naturais do Cerrado (ex: pequi)
Evitar itens inadequados (ex: madeira)
Objetivo: coletar corretamente os itens
♻️ 2. Reciclagem
Coleta de garrafas plásticas
Evitar que caiam no rio
Relacionado à reutilização (ex: criação de pufes com garrafas PET no projeto físico)
🔥 3. Combate a incêndios
Apagar focos de incêndio em tempo limitado
Inspirado nas queimadas ocorridas no Cerrado, incluindo a região de Brasília
🛠️ Tecnologias utilizadas
C#
Unity
Desenvolvimento para dispositivos mobile
🤝 Trabalho em equipe
Projeto desenvolvido em grupo, com divisão de responsabilidades entre os integrantes.
O desenvolvimento do jogo foi realizado individualmente, enquanto os demais membros contribuíram com outras partes do projeto no circuito de ciências.
